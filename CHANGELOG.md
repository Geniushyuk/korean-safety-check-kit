# Changelog

## v0.1.0

Initial submission-ready public release.

### Added

- 9 Korean safety-check scenarios covering phishing, smishing, sextortion, relationship safety, safe return, scam prevention, and parent protection.
- No-build JSON export at `data/scenarios.ko.json`.
- CommonJS scoring utilities in `src/index.js`.
- Scenario validation script and JSON schema.
- Smoke tests for scenario count, scoring levels, official resources, and share copy.
- Safe Korean wording guide.
- Integration examples for static pages, Discord bots, family-protection pages, and product onboarding.
- `AGENTS.md` for AI-assisted maintainer workflow.
