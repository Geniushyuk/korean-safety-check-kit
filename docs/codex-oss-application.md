# Codex for Open Source application notes

## Project
Korean Safety Check Kit is an open-source Korean safety-check dataset and scoring engine for lightweight public-interest safety UX.

## Planned Codex use
- Triage new scenario requests and classify them as copy fix, scoring bug, resource update, or new checklist.
- Review PRs for overclaiming, unsafe wording, missing emergency-resource guidance, and test coverage.
- Generate regression tests for scoring and recommended-action changes.
- Draft release notes for new Korean safety scenarios.

## Why this matters
The project targets Korean-language public-interest safety workflows where users may be under pressure: phishing calls, smishing links, sextortion threats, romance/investment scams, and family-protection scenarios.
